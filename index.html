Flexbox

O flexbox é um ferramenta muito interessante para responsividade semelhante aos groups do bubble

Pode-se alinhar tanto em colunas como em linhas (propriedade flex-direction), se não definidas limitações o displayflex oucupa todo o espaçamento da tela

  • justify = space-around = mesmo espaço entre os elementos

  • nth-child() = elementos de mesmo nome que podem ser selcioanados por uma número individualemente, container:nth-child(2)

  • flex-wrap: wrap; = se não cabe eu quebro a linha

  • flex-basis: 200px; = tamanho inicial

  • flex-grow: 1; = se não cabe eu cresço

  • flex-shrink: 1; = se cabe eu encolho

  • flex: 1 1 200px; = flex-grow, flex-shrink, flex-basis
